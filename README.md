# Triangle Matrix

## Описание
Triangle Matrix - это 2D shoot'em up игра, разработанная на HTML5 Canvas и JavaScript. Игрок управляет треугольником, сражается с различными геометрическими врагами (круги, квадраты, треугольники и боссы), подбирает хилки и проходит уровни. Игра включает 4 основных уровней и 2 босса.

## Особенности
- **Управление:** Движение с помощью клавиш WASD, стрельба на ЛКМ
- **Механики:** Здоровье (HP), щит, стрельба пулями, частицы эффектов
- **Враги:** Различные типы врагов с уникальным поведением
- **Уровни:** 6 уровней, 2 из которых это боссы
- **Обучение:** Встроенный туториал для новичков
- **Звук:** Звуковые эффекты для действий игрока

## Требования
- Python 3.x (для запуска сервера)
- Современный веб-браузер с поддержкой HTML5 Canvas

## Установка и запуск
1. Клонируйте или скачайте проект в папку.
2. Запустите сервер командой:
   ```
   python run_game.py
   ```
3. Браузер автоматически откроется на главной странице игры.

## Структура проекта
- `run_game.py` - Скрипт для запуска локального HTTP-сервера
- `html/index.html` - Главная страница с меню
- `css/style.css` - Общие стили
- `js/` - Общие скрипты (фон, звук)
- `character/Player.js` - Класс игрока
- `enemies/` - Классы врагов (Circle, Square, Triangle, Boss)
- `levels/1-6/` - Уровни игры, каждый с HTML, CSS и JS
- `ui_menu/` - Меню выбора уровней
- `playerHUD/` - Интерфейс игрока (HP, щит)
- `tutorial/` - Обучение игре

## Управление
- **WASD** - Движение
- **Мышь** - Прицеливание и стрельба
- **Щит** -  Активация на ПКМ

## Уровни
1. **Уровень 1** - Начало просто убить врагов
2. **Уровень 2** - Выживать определённое кол-во времени 
3. **Уровень 3** - Босс
4. **Уровень 4** - Побег
5. **Уровень 5** - Щиты
6. **Уровень 6** - Финальный босс

## Разработка
Игра написана на чистом JavaScript без внешних зависимостей. Для разработки можно редактировать файлы напрямую и перезагружать страницу.
